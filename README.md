## Log Analyser Using Python For *Linux*

Prerequisite:
<ol>
<li>Python 3.x (x: Version)</li>
<li>Any IDE</li>
</ol>

 Direction To use Code in ***Linux***:
 <ol>
 <li>Change the log file path wanted to view</li>
 <li>chmod +x lanalysis.py</li>
 <li>.\lanalysis.py</li>
</ol> 

 `OR`
 
<ol>
 <li>Change the log file path wanted to view</li>
 <li>sudo su</li>
 <li>python3 lanalysis.py</li>
</ol>
